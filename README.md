Crypto Facilities REST API v3
==================================
This is a sample RESTful application for [Crypto Facilities Ltd](https://www.cryptofacilities.com/), to demonstrate
the REST v3 API.


Getting Started
---------------

1. Amend the `cfRestApiV3Examples.js` script to enter your api keys and other details
1. Install the example dependencies with `$npm install`
1. Run the application with `$node cfRestApiV3Examples.js` after you check the example script actions

Functionality Overview
----------------------

* This application makes a REST call to every Crypto Facilities endpoint
